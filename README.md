# 🌿 KWLM Lab — 광합성 탐구 AI 학습 도구

중학교 2학년 과학 광합성 단원 수업에서 사용하는 AI 기반 KWLM 탐구 학습 앱입니다.  
학생들이 K·W·L·M 4단계를 순서대로 탐구하며 Claude AI와 1:1 대화를 통해 학습합니다.

---

## 📋 기능 요약

| 단계 | 내용 |
|---|---|
| **K** — 아는 것 | Claude AI와 소크라테스식 대화로 사전 오개념 진단 |
| **W** — 알고 싶은 것 | 질문 입력 → 학급 전체 워드클라우드 실시간 표시 |
| **L** — 배운 것 | 텍스트 설명 → Claude AI가 광합성 SVG 시각화 카드 생성 |
| **M** — 더 알고 싶은 것 | 심화 질문 → Claude AI가 탐구 과제 카드 추천 |
| **교사 대시보드** | 학생별 진행 현황, 오개념 키워드, 질문 카테고리 분포 모니터링 |

---

## 🚀 설치 방법 (3단계)

### 1단계 — 파일 다운로드

이 폴더(`kwml/`) 전체를 교실 PC에 복사합니다.  
별도의 설치나 프로그램이 필요하지 않습니다.

### 2단계 — Claude API 키 입력

`index.html` 파일을 메모장(또는 텍스트 편집기)으로 열고,  
파일 상단에 있는 아래 부분을 찾아 API 키를 입력합니다:

```html
window.KWLMLAB_CONFIG = {
    apiKey: '',             ← 여기에 API 키를 입력하세요
    teacherCode: '1234',    ← 교사 코드 (원하면 변경 가능)
    model: 'claude-sonnet-4-20250514'
};
```

**API 키 발급 방법:**
1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 후 로그인
3. 좌측 메뉴 → **API Keys** → **Create Key**
4. 생성된 키(`sk-ant-api03-...`)를 복사하여 붙여넣기

### 3단계 — 브라우저로 실행

`index.html` 파일을 **크롬 브라우저**로 엽니다.  
로그인 화면이 나타나면 준비 완료입니다!

> ⚠️ 인터넷 연결이 필요합니다 (Claude API 호출용).

## 👨‍🏫 수업 진행 방법

### 학생 사용 흐름

```
1. index.html 열기 (교사가 공유한 파일 또는 URL)
2. 이름, 반, 번호 입력 후 [탐구 시작하기] 클릭
3. K → W → L → M 순서로 탐구 진행
4. 각 단계 완료 버튼 클릭 시 다음 단계로 이동
```

### 교사 대시보드 접근

1. 로그인 화면 하단 **교사 코드** 입력란에 `1234` 입력
2. [교사 대시보드로 이동] 클릭
3. 학생 진행 현황, 오개념 키워드, 질문 분포 확인 가능

> 💡 교사 코드는 `index.html`의 `teacherCode` 값을 변경하여 수정할 수 있습니다.

---

## 💾 데이터 저장 안내

- 모든 학습 데이터는 **이 기기의 브라우저(localStorage)에만 저장**됩니다.
- 외부 서버로 전송되지 않아 학생 개인정보가 보호됩니다.
- 같은 기기에서 여러 학생이 순서대로 로그인하면 각자의 데이터가 구분 저장됩니다.
- 교사 대시보드는 동일 기기에서 로그인한 학생 데이터만 표시됩니다.

## ❓ 자주 묻는 질문

**Q. "API 키가 설정되지 않았습니다" 오류가 떠요.**  
A. `index.html` 파일을 열어 `apiKey: ''` 부분에 API 키를 정확히 입력했는지 확인하세요.

**Q. 학생 기록이 사라졌어요.**  
A. 브라우저 쿠키/캐시를 지우면 localStorage도 초기화됩니다. 수업 중에는 브라우저 데이터를 지우지 마세요.

**Q. 교사 코드를 바꾸고 싶어요.**  
A. `index.html`의 `teacherCode: '1234'` 부분에서 원하는 숫자로 변경하세요.

**Q. 모바일에서 사용할 수 있나요?**  
A. 교실 PC(1280px 이상 화면) 기준으로 최적화되어 있습니다. 모바일은 화면이 깨질 수 있습니다.
