# KWLM Lab — Claude Code 작업지시서

> 이 문서는 Claude Code에게 전달하는 단계별 개발 작업지시서입니다.
> 작업 전 반드시 프로젝트 루트의 `CLAUDE.md`를 먼저 읽고 규칙을 숙지하세요.

---

## 📌 프로젝트 개요

- **서비스명**: KWLM Lab
- **대상**: 중학교 2학년 과학 수업 (광합성 단원)
- **사용 환경**: 교실 PC, 크롬 브라우저
- **핵심 제약**: 서버 없음, 설치 없음, localStorage + Claude API만 사용

## ✅ MVP — 뼈대 + K단계 AI 챗봇

### 목표
앱이 실행되고, 로그인 후 K단계에서 Claude AI와 대화할 수 있는 상태

### 작업 목록

#### 1. 프로젝트 초기 세팅
- `kwlm-lab/` 폴더 생성
- `CLAUDE.md` 복사 (이미 작성된 파일)
- `index.html` 생성 (전체 앱 진입점, SPA 구조)
- `style.css` 생성 (초록 테마, CSS Variables 정의)
- `app.js` 생성 (라우터: 페이지 전환 함수, 현재 페이지 상태 관리)

#### 2. 로그인 페이지 (`pages/login.js`)
- 입력 필드: 이름, 반(1~6반 선택), 번호(1~35 선택)
- 교사 코드 입력란 (4자리 숫자, 맞으면 대시보드로 이동)
- 학생 로그인 시 `localStorage`에 `kwlmlab_student` 저장
- 이미 로그인된 학생이면 홈으로 자동 이동

#### 3. 네비게이션 메뉴 (`components/navbar.js`)
- 상단 고정 메뉴바
- 로고(🌿 KWLM Lab) + K·W·L·M 4단계 버튼
- 각 단계 버튼 클릭 시 해당 페이지로 이동
- 완료한 단계는 ✅ 표시, 현재 페이지는 강조(초록 배경)
- 우측에 학생 이름 + 로그아웃 버튼

#### 4. Claude API 공통 모듈 (`api/claude.js`)
- `callClaude(systemPrompt, messages, onStream)` 함수 작성
- 스트리밍 응답 지원 (답변이 타이핑되듯 표시)
- 에러 시 한국어 안내 메시지 반환
- API 키는 `index.html`의 `window.KWLMLAB_CONFIG.apiKey`에서 읽어옴

#### 5. 프롬프트 모음 (`data/prompts.js`)
- `PROMPTS.k` : K단계 시스템 프롬프트 (소크라테스식, 중2 수준, 광합성 전문)
- `PROMPTS.w` : W단계 프롬프트 (질문 분류용)
- `PROMPTS.l` : L단계 프롬프트 (SVG 시각화 코드 생성)
- `PROMPTS.m` : M단계 프롬프트 (탐구 과제 제안)

**K단계 프롬프트 내용 (작성 기준)**:
```
너는 중학교 2학년 광합성 수업을 돕는 친절한 탐험가야.
학생이 무언가를 말하면 절대 바로 정답을 알려주지 마.
대신 학생의 생각을 한 단계 더 발전시키는 질문을 하나만 해줘.
예: 학생이 "식물은 물만 먹고 자라요"라고 하면
→ "맞아, 물이 아주 중요하지! 그럼 물만 있으면 깜깜한 상자 안에서도 잘 자랄까?"
짧고 친근하게, 이모지를 가끔 사용해.
```

#### 6. K단계 페이지 (`pages/k-page.js`)
- 상단: "💡 내가 아는 것 — 광합성에 대해 아는 것을 자유롭게 말해보세요!" 안내문
- 채팅 UI: 말풍선 형태, 학생(오른쪽 초록), AI(왼쪽 흰색)
- 입력창 + 전송 버튼 (Enter 키도 동작)
- AI 응답 중 로딩 스피너 + "AI가 생각하는 중..." 표시
- 대화 내용 `localStorage`에 `kwlmlab_k_chat`으로 저장
- 하단에 "K단계 완료" 버튼 → 클릭 시 W단계로 이동 + 완료 상태 저장

#### 7. 홈 페이지 (간단 버전) (`pages/home.js`)
- 로그인 후 첫 화면
- "안녕하세요, [이름]님! 광합성 탐구를 시작해봐요 🌿" 환영 메시지
- K·W·L·M 4단계 카드 (완료 ✅ / 진행중 🔵 / 미시작 ⬜)
- 각 카드 클릭 시 해당 페이지로 이동

### MVP 완료 기준
- [ ] 브라우저에서 `index.html` 열면 로그인 화면이 뜬다
- [ ] 로그인 후 홈 화면으로 이동, 네비게이션 메뉴가 보인다
- [ ] K단계 클릭 시 챗봇 화면이 뜬다
- [ ] Claude AI와 실제로 대화가 된다 (API 키 입력 후)
- [ ] 새로고침해도 대화 내용이 유지된다

## ✅ v0.2 — W단계 + L단계

### 목표
학급 질문 공유(워드클라우드)와 광합성 시각화 카드 기능 완성

### 작업 목록

#### 1. W단계 페이지 (`pages/w-page.js`)
- 상단: "❓ 알고 싶은 것 — 광합성에 대해 궁금한 점을 입력해보세요!" 안내문
- 텍스트 입력창 + "질문 추가" 버튼
- 내 질문 목록 표시 (삭제 가능)
- 질문 저장: `localStorage`의 `kwlmlab_w_questions`에 배열로 저장
- 하단에 "W단계 완료" 버튼

#### 2. 워드클라우드 컴포넌트 (`components/wordcloud.js`)
- `localStorage`에 저장된 모든 학생의 W단계 질문을 읽어옴
- 키워드 빈도 분석 (조사 제거, 명사 추출 - 간단한 규칙 기반)
- SVG 또는 CSS로 워드클라우드 렌더링 (빈도 높을수록 크게)
- W단계 페이지 하단에 "우리 반 궁금한 것들 🔍" 섹션으로 표시
- 30초마다 자동 갱신

#### 3. L단계 페이지 (`pages/l-page.js`)
- 상단: "📖 배운 것 — 실험 후 알게 된 광합성 원리를 설명해보세요!" 안내문
- 텍스트 입력창 (여러 줄, 최소 2문장 유도)
- "시각화 카드 만들기" 버튼 클릭 시 Claude API 호출
- Claude에게 SVG 광합성 모식도 코드 생성 요청
- 생성된 SVG를 카드 형태로 렌더링하여 표시
- "내 카드 저장" 버튼 → `kwlmlab_l_card`에 저장
- 하단에 "L단계 완료" 버튼

**L단계 프롬프트 기준**:
```
학생이 광합성에 대해 텍스트로 설명했어. 
이 내용을 바탕으로 광합성 과정을 나타내는 SVG 모식도 코드를 만들어줘.
- 크기: viewBox="0 0 500 350"
- 포함 요소: 잎 모양, 햇빛 화살표, CO2 입력, O2 출력, 포도당 생성
- 학생 텍스트에서 언급한 내용을 강조 표시해줘
- 색상은 초록(#52b788), 노랑(#ffd166), 파랑(#06d6a0) 계열 사용
- SVG 코드만 반환, 다른 설명 없이
```

### v0.2 완료 기준
- [ ] W단계에서 질문 입력 및 저장이 된다
- [ ] 워드클라우드가 렌더링된다
- [ ] L단계에서 텍스트 입력 후 시각화 카드가 생성된다
- [ ] 생성된 SVG 카드가 화면에 표시된다

## ✅ v0.3 — M단계

### 목표
심화 질문 입력 → AI 탐구 과제 제안 기능 완성

### 작업 목록

#### 1. M단계 페이지 (`pages/m-page.js`)
- 상단: "🔍 더 알고 싶은 것 — 수업 후 생긴 심화 질문을 입력해보세요!" 안내문
- 텍스트 입력창
- "탐구 과제 추천받기" 버튼 → Claude API 호출
- AI 응답을 카드 형태로 표시:
  - 🏷️ 분류 태그 ('개념 보충' 또는 '실생활 전이')
  - 📋 탐구 제안 내용
  - 🔗 관련 키워드 태그
- 응답 저장: `kwlmlab_m_result`
- 하단에 "M단계 완료" 버튼

**M단계 프롬프트 기준**:
```
중학교 2학년 학생이 광합성 수업 후 심화 질문을 했어.
아래 형식으로 JSON 응답해줘:
{
  "category": "개념 보충" 또는 "실생활 전이",
  "summary": "질문을 한 줄로 요약",
  "proposal": "탐구 과제 제안 (3~4문장, 중2 수준)",
  "keywords": ["키워드1", "키워드2", "키워드3"],
  "action": "당장 해볼 수 있는 실천 방법 1가지"
}
JSON만 반환, 마크다운 없이.
```

### v0.3 완료 기준
- [ ] M단계에서 질문 입력 후 AI 탐구 과제 카드가 생성된다
- [ ] 분류(개념 보충/실생활 전이)가 태그로 표시된다
